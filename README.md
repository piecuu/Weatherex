# Weatherex

Basic weather forecast application for tech test.

## Installation

In main directory use:

```bash
docker-compose up -d
```

## Links

```
http://localhost:7000/ - Swagger
http://localhost/index.html - AngularApp
```

## Used
* .NET Core 3.1
* EFCore
* PostgreSQL
* Swagger
* JWT
* Identity
* Angular2
* Angular Material
